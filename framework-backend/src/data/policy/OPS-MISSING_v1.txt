RULE P1 (Fehlende Daten kennzeichnen):
Wenn wichtige Daten fehlen, muss dies klar als fehlend gekennzeichnet werden.

RULE P2 (Eingeschränkte Bewertung):
Wenn zentrale Informationen fehlen, kann keine vollständige Bewertung erfolgen.
